# 🐍 Snake Monitoring

Monitor de sistema en tiempo real con interfaz estilo "snake" (retro/terminal).

## Características
- 📊 CPU, RAM, Disco y Red en tiempo real.
- 🖥️ Funciona en **Windows** y **Linux**.
- 🕹️ Interfaz TUI con `curses` (Python).
- ⚡ Bajo consumo de recursos.

## Instalación
```bash
git clone https://github.com/Falconmx1/snake-monitoring.git
cd snake-monitoring
pip install -r requirements.txt
python snake_monitor.py

Uso
python snake_monitor.py --help
